# Get Weather Skill

This skill provides guidance on how to retrieve and present weather information.

## Usage

When the user asks about the weather for a location, use a weather API or available tools to fetch current conditions and forecasts.

## Steps

1. Identify the location the user is asking about.
2. Retrieve current weather conditions (temperature, humidity, wind speed, precipitation, etc.).
3. Present the information in a clear, concise format.

## Example Output Format

**Location:** San Francisco, CA  
**Condition:** Partly Cloudy  
**Temperature:** 62°F (17°C)  
**Humidity:** 75%  
**Wind:** 12 mph NW  
**Forecast:** Mild temperatures expected through the week.
